Data Structures from Scratch (C++)

This project implements fundamental data structures from scratch: Singly Linked List  and Binary Search Tree (BST).

Operations Implemented
- Linked List: Insert, Search, and Delete.
- Binary Search Tree: Insert, Search, and Delete.

 Complexity Analysis

| Data Structure | Operation | Average Case | Worst Case |
| :--- | :--- | :--- | :--- |
| Linked List | Search/Insert/Delete | O(n) | O(n) |
| BST| Search/Insert/Delete | O(log n) | O(n) |

 How to Run
1. Compile: `g++ linked_list.cpp -o list`
2. Run: `./list`

Sample Output
- Linked List: 10 -> 20 -> 30 -> NULL
- BST: Search 70: Found
